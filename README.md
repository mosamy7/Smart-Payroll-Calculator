# Smart-Payroll-Calculator
A professional HR tool for calculating employee benefits, overtime (day/night), and holiday pay with precision and a modern dark-mode UI.
# 🚀 حاسبة المستحقات والرواتب الذكية - Smart Payroll Tool

أداة ويب متطورة مصممة لمساعدة موظفي الموارد البشرية (HR) والموظفين في حساب مستحقاتهم بدقة وسهولة. تم بناء الأداة لتوفر تجربة مستخدم سلسة مع مراعاة أدق التفاصيل الحسابية.

### ✨ المميزات الرئيسية:
- **فصل دقيق للمستحقات:** حساب الإضافي النهاري، الليلي، الراحات الأسبوعية، والعطلات الرسمية كلٌ على حدة.
- **تخصيص القطاعات:** دعم الحساب بناءً على نوع القطاع (صناعي 7 ساعات / خدمي 8 ساعات).
- **واجهة عصرية (UI/UX):** تصميم أنيق مع أنيميشن احترافي ودعم كامل للـ Dark Mode.
- **تفاعلية كاملة:** صورة شخصية متحركة وتأثيرات بصرية عند التفاعل مع العناصر.
- **دقة متناهية:** عرض النتائج بالتنسيق المالي (0.00 ج.م).

### 🛠 التقنيات المستخدمة:
- **HTML5:** لهيكلة الموقع بشكل دلالي.
- **CSS3:** للتنسيق المتقدم والأنيميشن (Custom Properties, Flexbox, Keyframes).
- **JavaScript:** للعمليات الحسابية المنطقية وتغيير الوضع الليلي.
